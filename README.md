# STUDENT RANT API

### installation

`npm install`


### test

`npm run test`

### lint

`npm run lint`


### lint fix

`npm run lint:fix`

### start up mongodb database

```bash
$ npm run mongodb:start
```

### stop mongodb database

```bash
$ npm run mongodb:stop
```

### purge mongodb database (deletes the entire database)

```bash
$ npm run mongodb:purge:force
```
### start local server

`npm run server`
